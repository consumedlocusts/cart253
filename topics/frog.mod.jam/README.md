# Frogfrogfrog

A game about shooting aliens

[Play it!]( https://consumedlocusts.github.io/cart253/topics/frog.mod.jamdex.html)

- [Ideas](./frogfrogfrog-ideas.md)
- [Planning](./planning.md)

## Description

> This modification of the original frogfrogfrog has added multiple game modes that include a challengig search & destroy initializing mode, target elimintion phase, and a final boss battle to test the will of your x bound, stationary player.
> Instruction: click to start, hover and click over the larger animated (sprite gif) alien/figure to start play, eliminate all aliens (8 total) clicking over them with the scope,defeat the boss (impossible) and survive to win game, watch for green circular "goop" projectiles.
